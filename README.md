# Example

## Docker

### Docker Compose

Build and Run the `docker-compose.yml` file in a detacthed state.

```console
docker-compose up --build -d
```

Bring down the service

```console
docker-compose down
```

### Commands

List Docker processes

```console
docker ps -a
```

List Docker images

```console
docker images
```

Remove Docker images

```console
docker image rm <image id>
```
